## üö® TROUBLESHOOTING GUIDE

### "My phone number doesn't work"
‚Üí Check you replaced ALL instances of (403) 555-1234  
‚Üí Search files for "555-1234" to confirm

### "Forms aren't submitting"
‚Üí Confirm you replaced PLACEHOLDER with real Formspree ID  
‚Üí Check Formspree dashboard for submissions  
‚Üí Check spam folder for test emails

### "Images are broken"
‚Üí Confirm images folder uploaded correctly  
‚Üí Check image filenames match HTML (case-sensitive)  
‚Üí Check browser console for 404 errors

### "Site is slow on mobile"
‚Üí Ensure you're testing on real 3G/4G, not WiFi  
‚Üí Check images are webp format (not jpg/png)  
‚Üí Run Google PageSpeed Insights test

### "Google isn't showing my site"
‚Üí Takes 1-3 days for new sites to index  
‚Üí Submit sitemap in Search Console  
‚Üí Request indexing manually for homepage  
‚Üí Create a few backlinks (LinkedIn, directory listings)
