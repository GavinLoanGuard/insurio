Embeddable Quote Widget

Add a quote calculator directly to your website. Users get instant estimates without leaving your site. Leads are automatically tracked to your account.

Installation

Add this snippet before your closing </body> tag:

<!-- Insurio Quote Widget --> <script> (function(w,d,s,o,f,js,fjs){ w['InsuriWidget']=o;w[o]=w[o]||function(){ (w[o].q=w[o].q||[]).push(arguments)}; js=d.createElement(s);fjs=d.getElementsByTagName(s)[0]; js.src=f;js.async=1;fjs.parentNode.insertBefore(js,fjs); })(window,document,'script','insuri', 'https://cdn.insurio.ca/widget.js'); insuri('init', { partnerId: 'YOUR_PARTNER_ID' }); </script> <!-- Place widget container where desired --> <div id="insurio-widget"></div>

Configuration Options

insuri('init', { partnerId: 'YOUR_PARTNER_ID', theme: 'light', // 'light' or 'dark' primaryColor: '#f59e0b', defaultAmount: 500000, showBankComparison: true });

REST API

Full programmatic access to quotes, applications, and policy management. Build insurance directly into your product experience.

Base URL: https://api.insurio.ca/v1
Beta Access Available

Endpoints

POST /quotes

Generate a quote based on applicant details.

{ "mortgage_amount": 500000, "term_years": 25, "applicant": { "age": 35, "smoker": false, "province": "AB" }, "coverage_type": "life" }
GET /quotes/{quote_id}

Retrieve quote details and available carriers.

POST /applications

Submit a full application for underwriting.

GET /applications/{app_id}/status

Check application status and underwriting decision.

Webhooks

Subscribe to real-time events for application updates, policy issuance, and commission payouts.

quote.created application.submitted application.approved policy.issued commission.paid
